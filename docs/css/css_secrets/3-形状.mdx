# 形状

## 自适应的椭圆

### 椭圆

<div class="css-secret-3-basic css-secret-3-oval"></div>

```css
.oval {
    /* 水平半径 / 垂直半径 */
    border-radius: 50% / 50%;
}
```

### 半椭圆

<div class="css-secret-3-basic half-oval"></div>

```css
.half-oval {
    border-radius: 100% 0 0 100% / 50%;
}
```

### 四分之一椭圆

<div class="css-secret-3-basic quarter-oval"></div>

```css

    border-radius: 100% 0 0 0;
}
```

### 矫正内部变形

    <div>Hello</div>
</div>

```html
<div class="css-secret-3-basic parallel-1"><div>Hello</div></div>
```

```css
.parallel-1 {
    transform: skewX(-20deg);
}

.parallel-1 > div {
    transform: skewX(20deg);
}
```

### 用伪类矫正

```css
.parallel-2 {
    position: relative;
}

.parallel-2::before {
    content: '';
    position: absolute;
    inset: 0;
    z-index: -1;
    background: #f5bcbc;
    transform: skew(-20deg);

```

## 切角效果

<div class="css-secret-3-basic cut-corner"></div>

```css
.cut-corner {
    background: linear-gradient(-45deg, transparent 15px, #f5bcbc 0);
}
```

### 应用到四个角

<div class="css-secret-3-basic cut-all-corner"></div>

```css
.cut-all-corner {
    background: linear-gradient(135deg, transparent 15px, #f5bcbc 0) top left, linear-gradient(
                -135deg,
                transparent 15px,
                #f5bcbc 0
            ) top right,
        linear-gradient(-45deg, transparent 15px, #f5bcbc 0) bottom right, linear-gradient(
                45deg,
                transparent 15px,
                #f5bcbc 0
            ) bottom left;
    background-size: 51% 51%; /* 理论 50% 是够的 */
    background-repeat: no-repeat;
}
```
