## 更新说明—2022年8月30日

### 新页面“画廊”

主界面“画廊”现已开放，将展示四张近期的图片作品。

### 壁纸切换

现在主界面提供三张壁纸，可以点击右下角“切换壁纸”按钮选择你喜欢的壁纸。

### 其他新增特性

* 用户可以点击右上角头像来切换状态。

* “温馨提示”：由于页面用到许多现代 CSS 特性，需要用户使用较新的浏览器版本。游客在第一次进入页面时会受到一个浏览器限制的提示，引导用户更新浏览器版本。

### 错误修复

* 调整了背景动画，修复了回到主界面时可能会发生的元素抖动。

* 解决了因服务端渲染导致的图片加载问题。
