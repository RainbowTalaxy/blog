# Python 解释器

调用和退出解释器：

```shell
# 启动
python

# 退出
quit()
```

获取命令行参数：

```py
import sys
print(sys.argv)
```

Python 源码文件默认使用 UTF-8 编码。如果要使用其他编码，可以在文件开头写上特殊注释：

```py
# -*- coding: encoding -*-
```
